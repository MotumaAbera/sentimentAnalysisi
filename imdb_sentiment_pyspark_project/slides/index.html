Sentiment Analysis of IMDb Movie Reviews Using Apache Spark and PySpark Machine Learning

Group project presentation | Replace with group names

Project Goal

Classify IMDb movie reviews into positive and negative sentiment categories using a scalable PySpark machine learning workflow.

Input

Large CSV file of movie review text and sentiment labels.

Processing

Spark DataFrames, cleaning, tokenization, stopword removal, TF-IDF.

Output

Model comparison using accuracy, precision, recall, and F1-score.

Why Spark?

  • Distributed processing for large text datasets.
  • DataFrame API for scalable cleaning and aggregation.
  • ML pipelines for reproducible feature extraction and classification.
  • Same notebook can run locally or on a cluster.

Data Pipeline

Load CSV into Spark DataFrame
Clean review text and encode labels
Tokenize and remove stopwords
Extract TF-IDF features
Train and evaluate models

Models Compared

Logistic Regression

Strong baseline for sparse TF-IDF features.

Naive Bayes

Fast probabilistic model for text classification.

Decision Tree

Interpretable model used as a non-linear comparison.

Evaluation Metrics

  • Accuracy: overall percentage of correct predictions.
  • Precision: reliability of positive predictions.
  • Recall: ability to find actual positive reviews.
  • F1-score: balanced measure of precision and recall.

Expected Result Pattern

Logistic Regression: strongest expected TF-IDF performance
Naive Bayes: competitive and efficient baseline
Decision Tree: interpretable but weaker on sparse text

Key Insights

  • TF-IDF highlights words that distinguish positive and negative reviews.
  • Linear models are usually well suited to high-dimensional text vectors.
  • Errors may come from sarcasm, negation, mixed opinions, and plot-heavy reviews.

Limitations and Future Work

  • Bag-of-words features do not fully capture word order or context.
  • Future work could test n-grams, feature selection, hyperparameter tuning, or transformer embeddings.
  • The workflow can scale to larger datasets using a Spark cluster.

Conclusion

The project delivers a complete PySpark sentiment analysis pipeline for IMDb movie reviews, including scalable preprocessing, TF-IDF features, three machine learning classifiers, evaluation metrics, and visual results.
